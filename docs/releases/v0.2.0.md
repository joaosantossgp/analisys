# Release v0.2.0

Data: 2026-04-08

## Resumo

Este ciclo consolida a V2 web como um slice de produto mais robusto e mais
coerente visualmente, com foco em `Home -> Empresas -> Empresa`.

## Entregas principais

- estabilizacao do cliente web para falhas de transporte e upstream
- tratamento amigavel de erro na rota `/empresas`
- fallback parcial quando filtros falham sem derrubar a listagem
- consolidacao do Design System em `/design-system`
- adocao visual do sistema nas rotas `/`, `/empresas` e `/empresas/[cd_cvm]`
- padronizacao de loading, error e not-found
- limpeza de copy com encoding quebrado nas superficies mais visiveis
- integracao de video ambiente no topo da aplicacao, restrito a faixa superior
